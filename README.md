# Modern Fashion Store

> ⚡ Built with modern frontend technologies and best practices

A modern and responsive fashion ecommerce frontend built with Next.js, focused on delivering a clean UI, smooth animations, and a premium user experience.

## ✨ Features

- Fully responsive design (mobile, tablet, desktop)
- Clean and minimal UI inspired by modern fashion brands
- Product collection and category layout
- Smooth animations and transitions
- Reusable and scalable component structure

## 🛠️ Tech Stack

- **Next.js 13.5**
- **React 18**
- **Tailwind CSS**
- **Radix UI**
- **React Hook Form + Zod**
- **Framer Motion + GSAP**
- **Embla Carousel**
- **Recharts**
- **Lucide Icons**
- **Sonner (toast notifications)**

## 🚀 Getting Started

Clone the project and install dependencies:

```bash
git clone https://github.com/robertbunescu/modern-fashion-store.git
cd modern-fashion-store
npm install
npm run dev
```
